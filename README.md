This project presents a comprehensive **Bank Loan Report** built using **SQL Server Management Studio (SSMS)** and **Tableau**. It analyzes loan application trends, funding performance, and borrower characteristics to help stakeholders make data-driven decisions.

---

# Project Overview

The dashboard provides insights into:

- Loan application volumes (Total, MTD, PMTD)
- Funded amounts and repayments
- Interest rate and DTI (Debt-to-Income) trends
- Loan quality segmentation (Good vs Bad Loans)
- Loan status tracking
- Regional, term-based, and demographic breakdowns

# Business Objective

To help the bank understand:

- How loan performance varies by time, region, and borrower profile
- Which segments contribute to higher repayment rates
- Where risk is concentrated (e.g., bad loans, high DTI)
- How to optimize loan issuance and collections

- ## Key KPIs Tracked

- **Loan Volume**: Total, MTD, PMTD
- **Funded Amounts**: Total, MTD, PMTD
- **Amount Received**: Total, MTD, PMTD
- **Interest Rate**: Average, MTD, PMTD
- **DTI Ratio**: Average, MTD, PMTD
- **Loan Quality**: Good vs Bad Loans (volume, funding, repayment)
- **Loan Status**: Current, MTD, PMTD

---

## Dashboard Highlights

- **Monthly Trends**: Loan issuance over time
- **Regional Analysis**: State-wise loan performance
- **Loan Term & Purpose**: Impact on funding and repayment
- **Borrower Profile**: Employee length, home ownership, and more
